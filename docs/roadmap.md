# Roadmap

## Phase 1 (MVP)
- [ ] View, edit, delete, and enter shot data, including machines, beans, batches, and shot parameters
- [ ] Trigger automatic training of models when new shots data is entered
- [ ] Request predictions for shot yield, pressure, and time based on entered shot data
- [ ] Track when maintenance is due on machines and alert users
- [ ] Provide insights for grinding beans based on well established barista rules
- [ ] Add authentication and authorization
- [ ] Add user profiles
- [ ] Provide insights for pulling shots based on predictions and previous shot data

## Phase 2
- [ ] Provide insights for grinding beans based on well established barista rules
- [ ] Add mobile app support for shot logging and maintenance alerts
- [ ] Implement real-time collaboration features for shot data sharing
- [ ] Add advanced analytics dashboard with custom reports
- [ ] Integrate with coffee bean supplier APIs for automatic inventory updates

## Phase 3
- [ ] Implement IoT sensor integration for real-time machine monitoring
- [ ] Add AI-powered flavor profile matching and recommendation engine
- [ ] Create community platform for sharing shot recipes and techniques
- [ ] Implement predictive maintenance scheduling with machine learning
- [ ] Add enterprise features for multi-location café management
- [ ] Integrate with smart home devices for automated workflow triggers
- [ ] Track when maintenance is due on machines and alert users
